# Somnia API

## Authentication
### register
`Request Type: POST`
> /api/register

#### requirement
```
email: 'example@example.com'
```

#### returns
```
{
  "user": {
    "email": "example@example.com",
    "createdAt": "2016-05-21T10:14:04.867Z",
    "updatedAt": "2016-05-21T10:14:04.867Z",
    "id": "574034ec4bb5f9e40d050838"
  },
  "token": [token]
}
```

### login
`Request Type: POST`
> /api/login

#### requirement
```
email: 'example@example.com',
password: 'mySuperPassword'
```

#### returns
```
{
  "user": {
    "email": "example@example.com",
    "createdAt": "2016-05-21T10:14:04.867Z",
    "updatedAt": "2016-05-21T10:14:04.867Z",
    "id": "574034ec4bb5f9e40d050838"
  },
  "token": [token]
}
```
